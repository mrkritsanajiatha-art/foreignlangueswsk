## 1. ภาพรวมระบบ

ระบบนี้ถูกพัฒนาสำหรับ **กลุ่มสาระการเรียนรู้ภาษาต่างประเทศ** เพื่อ:

- ค้นหาครูที่ว่างสอนแทนเมื่อครูลา โดยอัตโนมัติจากตารางสอนและตารางเวร
- บันทึกสถิติการลาและการสอนแทนของครูแต่ละท่าน
- สร้างรูปภาพสรุปบันทึกการสอนแทนสำหรับแชร์ใน LINE
- แสดง Dashboard อันดับครูสอนแทนสูงสุดประจำภาคเรียน

**กฎการเลือกครูสอนแทน:**
1. ครูต้องไม่มีคาบสอนในคาบนั้น (`schedules`)
2. ครูต้องไม่มีเวรในคาบนั้น (`duties`)
3. เรียงลำดับ: จำนวนคาบสอนในวันนั้นน้อยก่อน → จำนวนสอนแทนสะสมน้อยก่อน → ชื่อ A-Z

## 6. Logic การทำงานหลัก

### 6.1 โหลดข้อมูลเริ่มต้น (`getInitialData`)

```
teachers table → รายชื่อครู, สถิติการลา/สอนแทน
วันในสัปดาห์ → hardcode ['จันทร์','อังคาร','พุธ','พฤหัสบดี','ศุกร์']
```

### 6.2 ค้นหาครูสอนแทน (`findSubstituteTeachers`)

```
1. ดึง schedules ของครูที่ลา ตามวันที่เลือก
2. สำหรับแต่ละคาบ:
   - ดึงครูทั้งหมด
   - กรอง: ไม่มีสอน AND ไม่มีเวรในคาบนั้น
   - เรียง: คาบสอนในวันนั้น (น้อย→มาก) → สอนแทนสะสม (น้อย→มาก) → ชื่อ
```

### 6.3 บันทึก (`saveAssignments`)

```
1. INSERT แต่ละ assignment → ตาราง substitutions
2. UPDATE leave_count + 1 → ครูที่ลา
3. UPDATE substitute_count + 1 → ครูแต่ละคนที่สอนแทน
4. ดึงสถิติใหม่ → อัปเดต Dashboard
```

### 6.4 สร้างรูปสรุป (`handleCapture`)

```
html2canvas(captureArea) → canvas → PNG dataURL
แสดง modal → กดค้างบันทึกบนมือถือ / ดาวน์โหลดบน PC
```
